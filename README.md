# Testlo-api

1. `yarn install`
2. clone the `.env.example` file y rename it as `.env`.
3. Use your environment variables
4. set up the database

```
docker-compose up -d
```

5. Put the server up
   `yarn start:dev`

6. Run the seeds to populate database
   `GET: http://localhost:3000/api/seed/products`
